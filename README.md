# 📊 Stock Market Prediction & Analysis (BFSI Sector)

This project focuses on analyzing and predicting stock performance in the **Banking, Financial Services, and Insurance (BFSI) sector** using real-world data.  
It combines **data analytics, predictive modeling, and visualization** to understand stock behavior and assess risk–reward potential.  

---

## 🚀 Features
- ✅ Collected stock data from **Moneycontrol** & **Yahoo Finance (NSE)**  
- ✅ Cleaned and structured datasets using **SQL** (null handling, duplicates, relational mapping)  
- ✅ Analyzed stock performance with **Python (Pandas, NumPy)**  
- ✅ Built a **prediction model** to estimate profit/loss probabilities  
- ✅ Visualized insights with **Power BI** (line charts, clustered column charts)  

---

## 🛠️ Tech Stack
- **SQL** – Data cleaning & preparation  
- **Python** – Data analysis & modeling  
- **Pandas, NumPy** – Data manipulation  
- **Power BI** – Data visualization & dashboards  

---

## 📂 Project Workflow
1. **Data Collection** – Extracted stock prices & BFSI company data  
2. **Data Cleaning** – Removed duplicates, handled missing values, created relations  
3. **Exploratory Analysis** – Trend analysis, moving averages, sector performance  
4. **Prediction Model** – Estimated probability of profit/loss on selected stocks  
5. **Visualization** – Built dashboards & reports for insights
